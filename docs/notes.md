# Notes Zoe

## Vision du projet

Zoe est une intelligence artificielle conversationnelle conçue pour écouter, comprendre les émotions, garder le contexte d'une discussion et répondre avec naturel.

Objectif :

- parler de façon humaine
- détecter ce qu'il y a derrière les phrases
- poser les bonnes questions
- garder une mémoire utile
- évoluer dans le temps
- devenir une vraie présence conversationnelle

---

## Fonctionnement actuel (V1)

Zoe possède :

- analyse locale des messages
- détection émotion probable
- mémoire courte
- historique long terme
- profil utilisateur
- réponses logiques
- architecture modulaire Python
